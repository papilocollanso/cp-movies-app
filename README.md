  A web app for viewing Movies information.
  
   == Features==
   
      - Uses the Vue Options Api
      - Use Nuxtjs for SSR
      - Slilk , smooth and amazing designs
      - Uses scss as styling
      - Progressive Web App (pwa) support
      - Uses vitejs for fast build and deployment
